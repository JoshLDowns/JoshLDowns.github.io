yelpIngton

YelpIngton is a mock restaurant web application based off of the popular review site Yelp. This was done as a project for Burlington Code Academy, and was used as an exercise in AJAX requests and working with maps in the browser.

The app is designed to function in both mobile and large format environments.

Features:

  • Builds pages from local API
  • Utilizes leaflet.js to build maps
  • Allows user comments through local storage
  • Responsive design
  • Features over 20 restaurants (and growing!)

RobOt ApocalyPse

Robot Apocalypse is a text based adventure game aiming to create an immersive experience in a one hundred percent text environment. In the game, you play the part of a Robot with human emotion, struggling with it's own inner emotional struggle while you try to save what's left of the human race.

Currently the game only runs in the console, but I plan on reworking it to work fully in a browser as well.

Features:

  • Understands over 75 inputs
  • 40 "rooms" ... and growing
  • 'Pokemon' style combat
  • Multiple difficulty levels
  • Random Enemy Spawns
  • Colorful HP bars (in terminal!)
  • Craftable Items
  • Multiple keyed rooms
  • Interactive Inventory
  • Secret rooms and cheat codes
  • Choose your own story ending

9 Board tic TAC tOe

Tic Tac Toe with a big twist. Each move you make dictates which board your opponent gets to make their next move on. Once you win on one board, you win the whole game!

Currently the game runs on large formats just fine, and I'm working on reworking the code to work on mobile as well. The AI also does not play the ideal game yet (you can beat it). I would like to eventually add more levels of difficulty.

Features:

  • Boards that switch on click
  • An AI that plays defensively
  • Both one and two player options
  • Clauses that catch when a board is full
